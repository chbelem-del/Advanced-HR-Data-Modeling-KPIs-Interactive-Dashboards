# 📊 HR Analytics Dashboard | Power BI

## 📌 Project Overview
This project presents an interactive Human Resources dashboard built with **Power BI** to help organizations monitor workforce performance, employee retention, turnover, and demographic trends.

The data model follows a **Star Schema** design and includes advanced **DAX measures**, interactive filtering, dynamic tooltips, and a professional multi-page dashboard.

---

## 🚀 Features

- ⭐ Star Schema data model
- 📈 HR KPIs (Headcount, Retention, Turnover)
- 👥 Workforce demographic analysis
- 🏢 Department & Job Level insights
- 🌍 Geographic employee distribution
- 🎛️ Interactive slicer panel with bookmarks
- 💡 Dynamic tooltips
- 🎨 Professional UI with multi-page navigation

---

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Star Schema Data Modeling

---

## 📊 Dashboard Pages

- **Home** – Landing page & navigation
- **Headcount** – Workforce overview and demographics
- **Retention** – Employee retention analysis
- **Turnover** – Employee turnover trends and termination insights
